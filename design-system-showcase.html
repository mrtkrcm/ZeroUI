🔧 Implementation Notes

From Go TUI to Web Components

This design system translates the terminal UI patterns from ConfigToggle into web-compatible components while maintaining the same visual hierarchy and interaction patterns.

Go/Lipgloss Original

titleStyle := lipgloss.NewStyle(). Foreground(lipgloss.Color("#FAFAFA")). Background(lipgloss.Color("#7D56F4")). Padding(0, 1). Bold(true) selectedStyle := lipgloss.NewStyle(). Foreground(lipgloss.Color("#7D56F4")). Bold(true) helpStyle := lipgloss.NewStyle(). Foreground(lipgloss.Color("#626262"))

CSS Translation

.terminal-title { color: #FAFAFA; background: #7D56F4; padding: 0.5rem 1rem; font-weight: 700; } .selected { color: #7D56F4; font-weight: 600; } .help-text { color: #626262; }
Key Design Principles:
  • Terminal-first aesthetics with monospace fonts for data
  • Purple color scheme matching the Charm library branding
  • Clear visual hierarchy with consistent spacing
  • Interactive states that feel responsive and immediate
  • Accessibility-first approach with proper contrast and focus states
